## v1.1 Requirements (All Complete)

### Discovery

- [x] **DISC-01**: User can search movies by title with real-time or submit-based filtering
  - **Outcome:** Validated - Implemented with 300ms debounce, case-insensitive partial match
  
- [x] **DISC-02**: User can filter movies by category or genre tag
  - **Outcome:** Validated - Category dropdown populated from API, exact match filtering
  
- [x] **DISC-03**: Search and filter results update the catalog view dynamically
  - **Outcome:** Validated - Combined filters work together, results count updates

### Account

- [x] **ACC-01**: User can register a new account from the public web UI
  - **Outcome:** Validated - Public /register route, no auth required
  
- [x] **ACC-02**: Registration form validates username uniqueness and password strength
  - **Outcome:** Validated - Email uniqueness checked, password min 8 chars
  
- [x] **ACC-03**: Newly registered users can immediately log in with their credentials
  - **Outcome:** Validated - Auto-login via token return, redirect to catalog
