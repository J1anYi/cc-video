# Milestone v1.5: Discovery Enhancement

**Status:** SHIPPED 2026-04-29
**Phases:** 13-14
**Total Plans:** 3

## Overview

v1.5 focused on content discovery features: personalized recommendations, continue watching, trending movies, and related movies.

## Phases

### Phase 13: Personalized Recommendations

**Goal**: Provide personalized movie recommendations based on user behavior
**Depends on**: Phase 8 (Favorites)
**Plans**: 2 plans

Plans:

- [x] 13-01: Recommendations Backend
- [x] 13-02: Recommendations Frontend

**Details:**
Implemented content-based filtering for recommendations:
- RecommendationService analyzes watch history and favorites
- Categories weighted 2x for favorites vs watched
- Continue watching shows movies with progress < 100
- Frontend components (ContinueWatching, Recommendations) in Catalog

### Phase 14: Trending & Discovery

**Goal**: Add trending movies and related movies features for enhanced content discovery
**Depends on**: Phase 13
**Plans**: 1 plan

Plans:

- [x] 14-01: Trending and Related Backend

**Details:**
Implemented trending and related movies:
- GET /trending returns most-watched movies in last 7 days
- GET /movies/{id}/related returns movies with same category
- Trending.tsx component with view count badges
- Integrated into Catalog page

---

## Key Decisions

- Decision: Content-based filtering by category (Rationale: Simple, effective, no ML needed)
- Decision: 7-day window for trending (Rationale: Balance freshness with data volume)
- Decision: Hide recommendations when filters active (Rationale: User intent is search, not discovery)
