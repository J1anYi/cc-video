# Requirements: CC Video

**Defined:** 2026-04-30
**Core Value:** Users can reliably browse the movie list and play administrator-uploaded videos through the web app.

## v1.8 Requirements

Requirements for v1.8 Content Organization milestone. Enhances social features with user watchlists and notification automation.

### User Watchlists

- [ ] **WATCHLIST-01**: User can create a themed watchlist
- [ ] **WATCHLIST-02**: User can add movies to their watchlist
- [ ] **WATCHLIST-03**: User can remove movies from their watchlist
- [ ] **WATCHLIST-04**: User can set watchlist as public or private
- [ ] **WATCHLIST-05**: User can view their own watchlists

### Watchlist Discovery

- [ ] **DISCOVER-01**: User can browse public watchlists
- [ ] **DISCOVER-02**: User can view watchlist details (owner, movies, title)
- [ ] **DISCOVER-03**: User can follow a watchlist

### Notification Automation

- [ ] **NOTIF-AUTO-01**: Notification created when followed user posts a review
- [ ] **NOTIF-AUTO-02**: Notification created when someone replies to user's comment
- [ ] **NOTIF-AUTO-03**: Notification created when user gains a new follower

### Activity Enhancements

- [ ] **ACTIVITY-ENH-01**: Activity shows when user adds a movie to favorites
- [ ] **ACTIVITY-ENH-02**: Activity shows when user creates a new watchlist

## v1.7 Requirements (Validated)

Requirements for v1.7 Social Extensions milestone. Extends v1.6 social features with user following, activity feed, and notifications.

### User Following

- [x] **FOLLOW-01**: User can follow another user
- [x] **FOLLOW-02**: User can unfollow another user
- [x] **FOLLOW-03**: User can view list of their followers
- [x] **FOLLOW-04**: User can view list of users they follow
- [x] **FOLLOW-05**: User can see follower/following count on profiles

### Activity Feed

- [x] **ACTIVITY-01**: User can view activity feed from followed users
- [x] **ACTIVITY-02**: Activity feed shows new reviews from followed users
- [x] **ACTIVITY-03**: Activity feed shows new ratings from followed users
- [x] **ACTIVITY-04**: Activity feed is sorted by most recent activity

### Notifications

- [x] **NOTIF-01**: User receives notification when followed user posts a review
- [x] **NOTIF-02**: User receives notification when someone replies to their comment
- [x] **NOTIF-03**: User can mark notifications as read
- [x] **NOTIF-04**: User can view notification history
- [x] **NOTIF-05**: Notification badge shows unread count

### Social Profile

- [x] **PROFILE-01**: User can view other users' public profiles
- [x] **PROFILE-02**: Profile shows user's reviews and ratings
- [x] **PROFILE-03**: Profile shows follower/following counts
- [x] **PROFILE-04**: Profile shows favorite movies (if public)

## v1.6 Requirements (Validated)

- [x] **RATING-01**: User can rate a movie 1-5 stars
- [x] **RATING-02**: User can update their rating
- [x] **RATING-03**: User can see average rating on movie
- [x] **REVIEW-01**: User can write a review for a movie

- [x] **REVIEW-03**: User can delete their review

## Out of Scope

| Feature | Reason |
|---------|--------|
| Email notifications | In-app notifications sufficient for v1.8 |
| Private profiles | All profiles public for social discovery |
| Block users | Defer to moderation milestone |
| Activity sharing | Social sharing deferred |
| Collaborative watchlists | Single owner for v1.8 |
