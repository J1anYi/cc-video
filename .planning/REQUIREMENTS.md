**Version:** 1.6

## Active Requirements

### SOC-01: Movie Ratings
**Priority:** High
**Phase:** 15

Users can rate movies with a star rating system.

**Acceptance Criteria:**
- [x] Users can rate movies 1-5 stars
- [x] Average rating displayed on movie cards and detail page
- [x] User's own rating shown when logged in
- [x] Rating can be updated by user
- [x] Rating count displayed alongside average

---

### SOC-02: Movie Reviews
**Priority:** High
**Phase:** 15

Users can write and read text reviews for movies.

**Acceptance Criteria:**
- [x] Users can submit text reviews for movies
- [x] Reviews displayed on movie detail page
- [x] Reviews show username and timestamp
- [x] Users can edit their own reviews
- [x] Users can delete their own reviews

---

### SOC-03: Comments on Reviews
**Priority:** Medium
**Phase:** 16

Users can comment on reviews to enable discussion.

**Acceptance Criteria:**
- [x] Users can add comments to reviews
- [x] Comments thread shown below review
- [x] Comments show username and timestamp
- [x] Users can delete their own comments

---

### SOC-04: Helpful Votes
**Priority:** Medium
**Phase:** 16

Users can mark reviews as helpful to surface quality content.

**Acceptance Criteria:**
- [x] "Helpful" button on each review
- [x] Helpful count displayed on review
- [x] One vote per user per review
- [x] Reviews sortable by helpfulness
