# Requirements: CC Video

**Defined:** 2026-04-30
**Core Value:** Users can reliably browse the movie list and play administrator-uploaded videos through the web app.

## v1.7 Requirements

Requirements for v1.7 Social Extensions milestone. Extends v1.6 social features with user following, activity feed, and notifications.

### User Following

- [ ] **FOLLOW-01**: User can follow another user
- [ ] **FOLLOW-02**: User can unfollow another user
- [ ] **FOLLOW-03**: User can view list of their followers
- [ ] **FOLLOW-04**: User can view list of users they follow
- [ ] **FOLLOW-05**: User can see follower/following count on profiles

### Activity Feed

- [ ] **ACTIVITY-01**: User can view activity feed from followed users
- [ ] **ACTIVITY-02**: Activity feed shows new reviews from followed users
- [ ] **ACTIVITY-03**: Activity feed shows new ratings from followed users
- [ ] **ACTIVITY-04**: Activity feed is sorted by most recent activity

### Notifications

- [ ] **NOTIF-01**: User receives notification when followed user posts a review
- [ ] **NOTIF-02**: User receives notification when someone replies to their comment
- [ ] **NOTIF-03**: User can mark notifications as read
- [ ] **NOTIF-04**: User can view notification history
- [ ] **NOTIF-05**: Notification badge shows unread count

### Social Profile

- [ ] **PROFILE-01**: User can view other users' public profiles
- [ ] **PROFILE-02**: Profile shows user's reviews and ratings
- [ ] **PROFILE-03**: Profile shows follower/following counts
- [ ] **PROFILE-04**: Profile shows favorite movies (if public)

## v1.6 Requirements (Validated)

- [x] **RATING-01**: User can rate a movie 1-5 stars
- [x] **RATING-02**: User can update their rating
- [x] **RATING-03**: User can see average rating on movie
- [x] **REVIEW-01**: User can write a review for a movie
- [x] **REVIEW-02**: User can edit their review
- [x] **REVIEW-03**: User can delete their review
- [x] **COMMENT-01**: User can comment on a review
- [x] **COMMENT-02**: User can edit their comment
- [x] **COMMENT-03**: User can delete their comment
- [x] **HELPFUL-01**: User can mark a review as helpful
- [x] **HELPFUL-02**: User can remove their helpful vote
- [x] **HELPFUL-03**: Reviews show helpful vote count

## Out of Scope

| Feature | Reason |
|---------|--------|
| Email notifications | In-app notifications sufficient for v1.7 |
| Private profiles | All profiles public for social discovery |
| Block users | Defer to moderation milestone |
| Activity sharing | Social sharing deferred |
