# UAT: Phase 164 - Cross-Platform Sync

## Test Cases

### TC-01: Device Registration
**Steps:**
1. Register mobile device
2. Register desktop device
3. Verify both devices appear in list

**Expected:** Both devices registered successfully

### TC-02: Watch History Sync
**Steps:**
1. Watch movie on mobile
2. Sync watch history
3. Check history on desktop
4. Verify movie appears

**Expected:** Watch history synced across devices

### TC-03: Playback Position Sync
**Steps:**
1. Start movie on mobile (50% progress)
2. Sync playback position
3. Open same movie on desktop
4. Verify progress shows 50%

**Expected:** Playback position synced

### TC-04: Preferences Sync
**Steps:**
1. Change preference on mobile
2. Sync preferences
3. Check preference on desktop
4. Verify value matches

**Expected:** Preferences synced correctly

### TC-05: Conflict Resolution
**Steps:**
1. Update same item on two devices offline
2. Sync both devices
3. Verify most recent version wins

**Expected:** Conflicts resolved correctly

## Acceptance Criteria
- Device registration works
- Watch history syncs
- Playback position syncs
- Preferences sync
- Conflicts resolved
