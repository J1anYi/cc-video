## Tasks

### Task 1: Create Quiz Models and Admin API

Goal: Database models and admin endpoints for quizzes
Files: backend/app/models/quiz.py, backend/app/routes/quiz.py
Implementation:
1. Create Quiz model with movie_id
2. Create QuizQuestion and QuizOption models
3. Admin POST endpoint for quiz creation
4. Admin POST endpoint for questions
Verification: Admin can create quizzes with questions

---

### Task 2: Implement Quiz Taking and Scoring

Goal: User quiz-taking functionality
Files: backend/app/routes/quiz.py
Implementation:
1. GET /api/quizzes/{id} - Get quiz (without correct answers)
2. POST /api/quizzes/{id}/attempt - Submit answers
3. Calculate score and return results
4. Show correct answers in response
Verification: User can take quiz and see score

---

### Task 3: Track Quiz Attempts in Profile

Goal: Store and display quiz attempts
Files: backend/app/models/quiz.py, frontend/src/pages/Profile.tsx
Implementation:
1. QuizAttempt model stores user attempts
2. GET /api/users/me/quizzes - User quiz history
3. Display attempts on profile page
Verification: Quiz attempts visible in profile

---

### Task 4: Create Quiz Leaderboards

Goal: Per-movie quiz leaderboards
Files: backend/app/routes/quiz.py, frontend/src/components/QuizLeaderboard.tsx
Implementation:
1. GET /api/quizzes/{id}/leaderboard endpoint
2. Rank users by score (ties by completion time)
3. QuizLeaderboard component
4. Display on quiz page
Verification: Leaderboard shows top scorers

---

## Execution Order
