## Tasks

### Task 1: Add Scheduling Fields to Movie Model

Goal: Extend Movie model with availability scheduling
Files: backend/app/models/movie.py
Implementation:
1. Add availability_start field (nullable datetime)
2. Add availability_end field (nullable datetime)
3. Add is_scheduled field (boolean, default false)
4. Create database migration
Verification: Model accepts new fields, migration runs cleanly

---

### Task 2: Implement Availability Background Task

Goal: Auto publish/unpublish content based on schedule
Files: backend/app/tasks/availability_manager.py
Implementation:
1. Create SchedulingService class
2. Implement scheduled task running every 5 minutes
3. Check movies with is_scheduled=true
4. Update visibility based on current time vs availability window
5. Send notifications for newly available content
Verification: Content auto-publishes at start time, auto-hides at end time

---

### Task 3: Add User Countdown and Scheduled Content Display

Implementation:
1. Add Coming Soon section for scheduled content

3. Show countdown on scheduled movie cards
4. Add Last Chance section for expiring content
Verification: Users see countdown, upcoming content displayed

### Task 4: Create Admin Calendar Management UI

Goal: Admin interface for managing content schedules
