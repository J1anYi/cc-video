# PLAN: Phase 25 - @Mentions

**Milestone:** v1.9 Admin & Safety
**Phase:** 25
**Goal:** Implement @mentions in comments and reviews

## Requirements

- MENTION-01: User can @mention other users in comments
- MENTION-02: User can @mention other users in reviews
- MENTION-03: Mentioned user receives notification
- MENTION-04: @mention links to mentioned user profile
- MENTION-05: @mention autocomplete suggests followed users

## Success Criteria

1. User can @mention in comments and reviews
2. Mentioned user gets notification with context
3. @username becomes clickable link to profile
4. Autocomplete suggests users while typing
5. Notifications for mentions work correctly

## Dependencies

- Existing Notification system
- Existing User model
