## Tasks

### Task 1: Implement Bulk Upload System

Goal: Allow admin to upload multiple videos at once
Files: backend/app/routes/movie.py, frontend/src/pages/admin/MovieUpload.tsx
Implementation:
1. Add bulk upload endpoint accepting multiple files
2. Create upload queue for sequential processing
3. Add progress tracking per file
4. Display upload progress in admin UI
Verification: Multiple files upload successfully with progress

---

### Task 2: Create Transcoding Pipeline

Goal: Transcode videos to multiple quality variants
Files: backend/app/models/video.py, backend/app/services/transcoding_service.py
Implementation:
1. Add VideoVariant model for quality variants
2. Create TranscodingService using FFmpeg
3. Implement background transcoding task
4. Update transcoding_status and progress
5. Generate HLS playlists for each quality
Verification: Videos transcoded to 1080p, 720p, 480p

---

### Task 3: Add Quality Selection to Player

Goal: Allow users to manually select video quality
Files: frontend/src/components/VideoPlayer.tsx
Implementation:
1. Create QualitySelector component
2. Parse master playlist for available qualities
3. Allow manual quality override
4. Store user preference in localStorage
Verification: User can select quality, preference persists

---

### Task 4: Implement Video Replacement

Goal: Replace video file while preserving metadata
Files: backend/app/routes/movie.py
Implementation:
1. Add replace_video endpoint
2. Delete old video files and variants
3. Upload new video file
4. Trigger re-transcoding
5. Preserve all movie metadata
Verification: Video replaced, metadata intact
