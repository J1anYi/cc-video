# UAT: Phase 167 - Advanced Video Tools

## Test Scenarios

### Test 1: Video Edit Project
**Steps:**
1. POST /video-tools/projects with movie_id and name
2. Verify project created with draft status
3. GET /video-tools/projects to list projects

**Expected:** Project created and listed

### Test 2: Thumbnail Generation
**Steps:**
1. POST /video-tools/thumbnails/{movie_id}/generate
2. Verify thumbnails generated at intervals
3. GET /video-tools/thumbnails/{movie_id}
4. Verify list of thumbnails returned

**Expected:** Thumbnails generated and listed

### Test 3: Chapter Management
**Steps:**
1. POST /video-tools/chapters with movie_id, title, start_time
2. Create multiple chapters
3. GET /video-tools/chapters/{movie_id}
4. Verify chapters listed in order

**Expected:** Chapters created and ordered

### Test 4: Audio Tracks
**Steps:**
1. POST /video-tools/audio-tracks with language info
2. Add multiple language tracks
3. GET /video-tools/audio-tracks/{movie_id}
4. Verify all tracks listed

**Expected:** Audio tracks added and listed

### Test 5: Subtitles
**Steps:**
1. POST /video-tools/subtitles with language and URL
2. Add subtitles for multiple languages
3. GET /video-tools/subtitles/{movie_id}
4. Verify all subtitles listed

**Expected:** Subtitles added and listed

### Test 6: Templates
**Steps:**
1. POST /video-tools/templates with name, category, config
2. Create public template
3. GET /video-tools/templates
4. Verify template listed

**Expected:** Template created and listed

## Status: PASS
All UAT scenarios completed successfully.
