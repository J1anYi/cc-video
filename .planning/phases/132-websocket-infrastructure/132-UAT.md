# UAT: Phase 132 - WebSocket Infrastructure

## Test Cases

### 1. WebSocket Connection

#### TC-WS-001: Connect with Valid Token
**Steps:**
1. Get JWT token via login
2. Connect to ws://localhost:8000/ws?token={token}
3. Verify connection_ack message received

**Expected:** Connection established

#### TC-WS-002: Connect with Invalid Token
**Steps:**
1. Connect with invalid token
2. Verify connection closed with code 4001

**Expected:** Connection rejected

### 2. Room Management

#### TC-WS-003: Join Room
**Steps:**
1. Connect with valid token
2. Send: {"type": "join_room", "room_id": "room1"}
3. Verify user_joined broadcast to others

**Expected:** Joined room successfully

#### TC-WS-004: Leave Room
**Steps:**
1. Join a room
2. Send: {"type": "leave_room", "room_id": "room1"}
3. Verify user_left broadcast

**Expected:** Left room successfully

### 3. Collaboration

#### TC-WS-005: Typing Indicator
**Steps:**
1. Two users join same room
2. User A sends: {"type": "typing", "room_id": "room1", "is_typing": true}
3. User B receives typing_start message

**Expected:** Typing indicator received

#### TC-WS-006: Watch Party Sync
**Steps:**
1. Two users join same room
2. User A sends: {"type": "watch_party_sync", "room_id": "room1", "action": "play", "timestamp": 120.5}
3. User B receives sync message

**Expected:** Sync message received

### 4. Heartbeat

#### TC-WS-007: Heartbeat Response
**Steps:**
1. Connect with valid token
2. Send: {"type": "heartbeat"}
3. Verify heartbeat response

**Expected:** Heartbeat acknowledged

### 5. Polling Fallback

#### TC-WS-008: Notification Polling
**Steps:**
1. Send GET to /poll/notifications?last_id=0
2. Verify response format

**Expected:** Notifications returned

#### TC-WS-009: Presence Polling
**Steps:**
1. Send GET to /poll/presence/room1
2. Verify present_users and typing_users

**Expected:** Presence data returned

## Test Execution

```bash
# Start backend
cd backend
uvicorn app.main:app --reload

# Connect with wscat
wscat -c "ws://localhost:8000/ws?token=YOUR_TOKEN"

# Check stats
curl http://localhost:8000/ws/stats
```
