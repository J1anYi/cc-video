# UAT: Phase 131 - GraphQL API

## Test Cases

### 1. GraphQL Endpoint

#### TC-GQL-001: Access GraphQL Playground
**Steps:**
1. Start application
2. Navigate to http://localhost:8000/graphql
3. Verify GraphiQL interface loads

**Expected:** GraphiQL playground visible

#### TC-GQL-002: Query Movies
**Steps:**
1. Execute query:
```graphql
query {
  movies(limit: 5) {
    id
    title
    genre
    averageRating
  }
}
```
2. Verify response contains movie data

**Expected:** Movie list returned

#### TC-GQL-003: Query Single Movie
**Steps:**
1. Execute query:
```graphql
query {
  movie(movieId: 1) {
    id
    title
    description
    releaseYear
  }
}
```

**Expected:** Single movie details returned

### 2. Query Operations

#### TC-GQL-004: User Query
**Steps:**
1. Execute query:
```graphql
query {
  user(userId: 1) {
    id
    username
    email
  }
}
```

**Expected:** User data returned

#### TC-GQL-005: Movie Search
**Steps:**
1. Execute query:
```graphql
query {
  searchMovies(query: "test") {
    id
    title
  }
}
```

**Expected:** Matching movies returned

### 3. Mutation Operations

#### TC-GQL-006: User Registration
**Steps:**
1. Execute mutation:
```graphql
mutation {
  register(input: {
    username: "testuser"
    email: "test@example.com"
    password: "testpass123"
  }) {
    id
    username
    email
  }
}
```

**Expected:** New user created

#### TC-GQL-007: User Login
**Steps:**
1. Execute mutation:
```graphql
mutation {
  login(username: "testuser", password: "testpass123")
}
```

**Expected:** JWT token returned

#### TC-GQL-008: Create Rating
**Steps:**
1. Login to get token
2. Execute mutation with auth:
```graphql
mutation {
  createRating(input: {movieId: 1, rating: 4.5}) {
    id
    rating
  }
}
```

**Expected:** Rating created

### 4. Filtering

#### TC-GQL-009: Filter Movies by Genre
**Steps:**
1. Execute query:
```graphql
query {
  movies(filter: {genre: "Action"}) {
    id
    title
    genre
  }
}
```

**Expected:** Only Action movies returned

#### TC-GQL-010: Filter Movies by Rating
**Steps:**
1. Execute query:
```graphql
query {
  movies(filter: {minRating: 4.0}) {
    id
    title
    averageRating
  }
}
```

**Expected:** Movies with rating >= 4.0 returned

## Test Execution

```bash
# Start backend
cd backend
uvicorn app.main:app --reload

# Access GraphQL playground
# Open http://localhost:8000/graphql in browser

# Test introspection
curl -X POST http://localhost:8000/graphql \
  -H "Content-Type: application/json" \
  -d '{"query": "{ __schema { types { name } } }"}'
```
