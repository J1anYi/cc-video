   - from app.services.video_file import video_file_service

2. Define VideoStreamingService class with stream method:
   ```python
   class VideoStreamingService:
       async def stream(self, video_file) -> FileResponse:
           file_path = Path(video_file.file_path)
           if not file_path.exists():
               raise HTTPException(status_code=404, detail="Video file not found")
           return FileResponse(
               path=file_path,
               media_type=video_file.mime_type,
               filename=video_file.filename
           )

   video_streaming_service = VideoStreamingService()
   ```

Note: FastAPI's FileResponse automatically handles Range requests and returns 206 Partial Content when the client sends a Range header. This enables seeking in browser video players.
