# Phase 117: User Groups - UAT

**Date:** 2026-05-01
**Tester:** Automated

## Test Cases

### UAT-01: Create Group
**Steps:**
1. Create a public group
2. Verify group is created with correct fields
3. Verify creator is added as owner

**Expected:** Group created successfully, creator is owner

### UAT-02: Join Public Group
**Steps:**
1. Create a public group as user A
2. Join group as user B
3. Verify user B is added as member

**Expected:** User B can join, appears in members list

### UAT-03: Private Group Access
**Steps:**
1. Create a private group as user A
2. Try to join as user B (should fail)
3. Invite user B as user A
4. Verify user B is now a member

**Expected:** Cannot join directly, can join via invite

### UAT-04: Group Collections
**Steps:**
1. Create a group
2. Create a collection within the group
3. Add a movie to the collection
4. Verify item appears in collection

**Expected:** Collection created, movie added successfully

### UAT-05: Group Discussions
**Steps:**
1. Create a group
2. Create a discussion thread
3. Reply to the discussion
4. Verify reply appears

**Expected:** Discussion and reply created successfully

### UAT-06: Activity Feed
**Steps:**
1. Create a group
2. Join the group
3. Create a discussion
4. Check activity feed
5. Verify all activities appear

**Expected:** Activity feed shows group_created, member_joined, discussion_created

### UAT-07: Member Role Management
**Steps:**
1. Create a group as owner
2. Invite another user
3. Promote user to admin
4. Verify role is updated

**Expected:** Role updated to admin

### UAT-08: Permission Checks
**Steps:**
1. Create a group as user A
2. Try to update group as user B (non-member)
3. Verify 403 Forbidden

**Expected:** Non-members cannot update group
