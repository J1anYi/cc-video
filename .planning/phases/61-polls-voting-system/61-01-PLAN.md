## Tasks

### Task 1: Create Poll Models and Admin API

Goal: Database models and admin endpoints for polls
Files: backend/app/models/poll.py, backend/app/routes/poll.py
Implementation:
1. Create Poll model with movie_id, title, expires_at
2. Create PollOption model linked to Poll
3. Create PollVote model with unique constraint
4. Admin POST/PUT/DELETE endpoints for polls
Verification: Admin can create polls with options

---

### Task 2: Implement Voting and Results API

Goal: User voting and results retrieval
Files: backend/app/routes/poll.py
Implementation:
1. POST /api/polls/{id}/vote endpoint
2. Check for duplicate votes
3. GET /api/polls/{id}/results endpoint
4. Calculate percentages and totals
Verification: User can vote, results show correctly

---

### Task 3: Add Poll Display to Frontend

Goal: Show polls on movie pages and community section
Files: frontend/src/components/PollWidget.tsx, frontend/src/pages/MovieDetail.tsx
Implementation:
1. Create PollWidget component
2. Add to MovieDetail page
3. Create Community polls page
4. Show active vs expired polls
Verification: Polls visible on movie pages

---

### Task 4: Implement Poll Notifications

Goal: Notify users of new polls on followed movies
Files: backend/app/services/poll_notification_service.py
Implementation:
1. Create PollNotificationService
2. Trigger on poll creation
3. Query users following the movie
4. Send new_poll notifications
Verification: Followers receive poll notifications

---
