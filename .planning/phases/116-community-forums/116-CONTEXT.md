# Phase 116: Community Forums

## Goal
Implement discussion forums per content category with thread creation, replies, and moderation.

## Requirements
- CF-01: Discussion forums per content category
- CF-02: Thread creation and replies
- CF-03: Forum moderation tools
- CF-04: Pinned and featured threads
- CF-05: Forum search and filtering

## Scope

### Backend
- Forum model for categories
- ForumThread model for discussions
- ForumPost model for replies
- ForumModeration model for moderation actions
- Forum service and routes

### Frontend
- Forum list component
- Thread view component
- Post reply component

## Success Criteria
- Forums organized by category
- Users can create threads and replies
- Moderation tools available
- Pinned threads support
- Forum search works
