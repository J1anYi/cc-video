# UAT: Phase 127 - Database Optimization

## Test Cases

### 1. Connection Pool

#### TC-DBO-001: Verify Pool Configuration
**Steps:**
1. Start application
2. Check pool size configuration in logs
3. Make multiple concurrent requests

**Expected:** Pool handles connections efficiently

#### TC-DBO-002: Pool Statistics
**Steps:**
1. Send GET request to `/db/stats`
2. Verify pool_size, checked_in, checked_out values

**Expected:** Pool statistics returned

### 2. Read Replicas

#### TC-DBO-003: Read Replica Routing
**Steps:**
1. Configure DATABASE_READ_REPLICA_URL
2. Use `get_read_db` dependency
3. Verify queries route to replica

**Expected:** Read queries use replica connection

### 3. Query Cache

#### TC-DBO-004: Cache Hit
**Steps:**
1. Execute cached query
2. Execute same query again
3. Check cache statistics

**Expected:** Second query hits cache

#### TC-DBO-005: Cache Invalidation
**Steps:**
1. Execute cached query
2. Invalidate cache with prefix
3. Execute query again

**Expected:** Cache miss after invalidation

### 4. Query Monitoring

#### TC-DBO-006: Slow Query Detection
**Steps:**
1. Execute slow query (complex join)
2. Check `/db/stats` for slow_queries

**Expected:** Slow query logged and tracked

#### TC-DBO-007: Query Statistics
**Steps:**
1. Make multiple API calls
2. Check `/db/stats` for query count

**Expected:** total_queries incremented

### 5. Database Health

#### TC-DBO-008: Health Check
**Steps:**
1. Send GET to `/db/health`
2. Verify status and latency

**Expected:** Status healthy, latency in ms

#### TC-DBO-009: Index Listing
**Steps:**
1. Send GET to `/db/indexes`
2. Verify index list returned

**Expected:** Indexes for all tables

#### TC-DBO-010: Table Analysis
**Steps:**
1. Send POST to `/db/analyze`
2. Verify tables analyzed

**Expected:** All tables analyzed

## Test Execution

```bash
# Database stats
curl http://localhost:8000/db/stats

# Database health
curl http://localhost:8000/db/health

# List indexes
curl http://localhost:8000/db/indexes

# Analyze tables
curl -X POST http://localhost:8000/db/analyze
```
